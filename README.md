# Customer-Shopping-Behavior-Analysis

Project Overview

This project focuses on analyzing customer shopping behavior using a combination of SQL, Python, and Power BI. The goal is to extract meaningful insights from customer transaction data to understand purchasing patterns, customer preferences, and business performance.

The project follows an end-to-end data analytics workflow:

Data Cleaning & Preparation (SQL + Python)
Exploratory Data Analysis (Python)
Business Insights & Visualization (Power BI)
🎯 Business Objective
Understand customer purchasing patterns
Identify high-value customers and products
Analyze sales trends and customer behavior
Support data-driven business decisions
🗂️ Dataset Description

The dataset contains customer shopping-related data such as:

Customer ID
Product details
Purchase behavior
Transaction values
Quantity purchased
Time-based attributes
